Tomcat 서버 이미 가동중일 때, 이클립스 서버와 비동기화 되있는 경우

프로세스를 강제로 종료해줘야 한다.

이 때 pid확인하는 법 (윈도우 기준)



명령 프롬프트에 다음 명령어 입력
```
netstat -a -n -o -p tcp
혹은
netstat -anop tcp
```

![img](./img/image.png)

톰캣에서 기 사용하고있는 포트 (8005 v 8080)의 pid를 체크한다.

해당 포트의 pid는 16648인 것을 확인

```
taskkill /f /pid 16648
```

프로세스 종료 명령어이다.

pid(프로세스 아이디)를 체크후, 종료할 맨 마지막 부분에 pid를 입력해준다.

(16648부분에 pid를 입력해준다.)
